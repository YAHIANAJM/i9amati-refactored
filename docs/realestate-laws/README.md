# Moroccan Real Estate Law — Reference Documentation

This folder documents the Moroccan legal framework governing co-ownership (copropriété) as it applies to i9amati. The primary legislation is **Loi n° 18-00** (promulgated October 3, 2002) as amended and completed by **Loi n° 106-12** (promulgated April 27, 2016, Dahir 1-16-49).

The full raw legal text (Bulletin Officiel n° 6514) is in [`../law.md`](../law.md).
